# README GUI 双图预览设计

## 目标

在 GitHub README 的“界面预览”章节中展示应用从启动到可用的完整体验。启动页位于主工作台之前，使阅读顺序与实际产品流程一致。

## 展示结构

采用纵向双图布局，不使用并排缩略图，保证桌面端和移动端都能清楚阅读界面文字：

1. “启动页面”图注及启动页截图。
2. “主工作台”图注及现有主工作台截图。

两张图片继续使用居中、`width="100%"` 的 HTML 图片标签，并提供准确的替代文本。
